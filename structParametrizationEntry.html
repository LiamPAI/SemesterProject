Detailed Description

Container for a complete data entry containing MeshParametrization.

Stores a data entry of type MeshParametrization along with the displacement vectors, energy difference, and the vector of tags

Local Ordering Convention:

  • Displacements: (4 x 2) for single-branch parametrizations, and (4n x 1) for multi-branch parametrizations Given the semantic equivalence between MeshParametrizationData and ParametrizationPoints, the displacements matrix must be ordered the exact same as described for PointEntry
