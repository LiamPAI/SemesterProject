Detailed Description

Class for computing linear finite element stiffness matrices.

Implements stiffness matrix computation for linear elastic problems using either plane stress or plane strain formulation

Eigen::Matrix< double, 8, 8 > LinearMatrixComputation::LinearFEElementMatrix::Eval ( const lf::mesh::Entity & cell)

Computes element stiffness matrix for given cell.

The returned matrix is of size 8x8 as a quadrilateral cell will have 4 nodes, so 8 entries due to the 2D nature of linear elasticity. For triangular cells, the final 2 rows and 2 columns are simply 0

cellMesh entity (element) to compute matrix for

30

32 case lf::base::RefEl::kTria(): {

34

43

45 // It is usually size 3x6 for triangles, although here it is 3x8 to account for return type of 8x8

50

52 elem_mat += area * B.transpose() * D * B;
53 break;
54 }
