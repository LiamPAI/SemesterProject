Detailed Description

Class for computing parametric finite element stiffness matrices.

Implements stiffness matrix computation for second-order elements using either plane stress or plane strain formulation

Computes element stiffness matrix for given cell.

The returned matrix is of size 18x18 as a quadrilateral cell will have 9 nodes, so 18 entries due to the 2D nature of linear elasticity. For triangular cells, the final 6 rows and 6 columns are simply 0 since second-order triangular cells have 6 nodes
